# 法条笔记助手

> 一款为法硕备考学生设计的桌面工具，聚合展示法条原文、司法解释、考点，并支持一键导出 Markdown 笔记。

## 背景与痛点
法硕考生查一条法条平均需翻阅3处资料，耗时8分钟。现有工具要么信息过载，要么与备考脱节。

## 功能亮点
- 🔍 秒级检索：输入法条编号，模糊匹配
- 📚 聚合展示：原文、司法解释、考点三标签页同屏
- 💾 一键导出：生成 Markdown 笔记，适配 Obsidian/Notion
- 🔌 离线可用：本地 JSON 数据，打包为 exe
- 🌐 在线补充（预研）：调用大模型 API 获取长尾法条

## 技术栈
- Python + Tkinter + ttkbootstrap
- 本地 JSON 数据存储
- PyInstaller 打包
- 通义法睿 API（预研）

## 快速开始
1. 下载 [最新 release](链接) 中的 exe 文件
2. 双击运行，无需安装
3. 输入法条编号（如“584”），点击搜索
4. 查看内容，点击“导出笔记”

## 数据策略
- 核心 20 条法条人工审核，准确率 100%
- 长尾法条通过 API 补充（V1.1 预研）
